MATERIAS APROBADAS

  • DESARROLLO DE LOGICA DE PROGRAMACION

    • Estudia los fundamentos del pensamiento lógico y estructurado para la resolución de problemas mediante algoritmos y diagramas de flujo, base para aprender cualquier lenguaje de programación

  • CONFIGURACION DE REDES INFORMATICAS

    • Se centra en el diseño, instalación y administración de redes de computadoras, incluyendo protocolos, topologías y dispositivos de comunicación.

  • PROGRAMACION DE BASES DE DATOS

    • Aborda la creación y gestión de bases de datos mediante lenguajes como SQL, aplicando consultas, procedimientos y buenas prácticas de almacenamiento de información

  • ANALISIS Y DISEÑO DE SISTEMAS INFORMATICOS

    • Enseña metodologías para estudiar necesidades de usuarios y diseñar soluciones tecnológicas eficientes, desde la planificación hasta la arquitectura del sistema.

  • DISEÑO DE PAGINAS WEB

    • Se enfoca en la creación de sitios web funcionales y atractivos utilizando lenguajes como HTML, CSS y JavaScript, aplicando principios de usabilidad y diseño responsivo.
